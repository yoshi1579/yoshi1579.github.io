4. What's my experience of the class?

This week we learned how allocator, such as vector, works in C++. If we use new with pre-defined type, then it takes O(1). However if we use it with user-defined type, it will take O(n) since it calls constructor n times. If vector is used to construct an array with user-defined type, it won't call constructor n times, but instead it will simply give us enough space from heap.
